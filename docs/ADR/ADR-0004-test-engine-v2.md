# ADR-0004: Test Engine v2

## Status
Accepted

## Context
Current test behavior is static and does not maximize learning quality.
The frozen architecture requires adaptive learning and fragrance-specific evaluation dimensions.

## Decision
Adopt Test Engine v2 as a dedicated learning system.
It uses dynamic dimensions from canonical fragrance attributes and stage-aware adaptive selection.

## Consequences
1. Test flow is decoupled from recommendation catalogs.
2. Hardcoded universal question sets are deprecated.
3. Learning quality and explainability improve through richer events.
