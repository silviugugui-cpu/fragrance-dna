# ADR-0005: Recommendation Engine v2

## Status
Accepted

## Context
Recommendation must be objective-driven, context-aware, and independent from test internals.

## Decision
Adopt Recommendation Engine v2 consuming only canonical contracts:
Persistent User DNA, context, objective, state projections, and canonical fragrance intelligence.

## Consequences
1. Recommendation behavior can evolve independently from test behavior.
2. Every recommendation requires explainability and reproducibility metadata.
3. Internal test implementation details are excluded from recommendation contracts.
