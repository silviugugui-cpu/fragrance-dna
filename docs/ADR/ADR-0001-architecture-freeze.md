# ADR-0001: Architecture Freeze

## Status
Accepted

## Context
FragranceDNA Canonical Architecture v2 has been approved and frozen.
Uncontrolled architectural drift would increase delivery risk and long-term maintenance cost.

## Decision
Adopt Canonical Architecture v2 as implementation law.
Architecture changes are exceptional and require ADR approval.

## Consequences
1. Delivery focuses on phased implementation, not redesign.
2. Architecture exceptions require explicit rationale, trade-offs, and rollback.
3. Governance and documentation become mandatory release artifacts.
